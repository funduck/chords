Directives: copyright
=====================

`{copyright: `*year*` `*owner*`}`

Copyright information for the song.

Examples:

    {copyright: 2014 Shitting Bull Inc.}
    {meta: copyright 2014 Shitting Bull Inc.}

See also: [meta](./directives-meta.md).

Directives: titles
==================

**Deprecated.** Use the [configuration files](https://www.chordpro.org/chordpro/chordpro-configuration/) instead.

`{titles:` *flush* `}`

If *flush* is `left`, song titles are printed to the left of the page.  
If *flush* is `right`, song titles are printed to the right of the page.  
If *flush* is `center`, titles are printed centered on top of the page. This is the default if this directive is not used.

Example:

`    {titles: center}`

Directives: album
=================

This directive defines an album this song occurs on. Multiple albums can
be specified using multiple directives.

Examples:

    {album: Songs Of Love And Hate}
    {meta: album Songs Of Love And Hate}

See also: [meta](./directives-meta.md).

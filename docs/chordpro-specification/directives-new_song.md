Directives: new\_song
=====================

Abbreviation: `ns`.

This directive indicates that the current song, if any, is complete and
that a new song will follow. This is implied at the start of a ChordPro
file.

Examples:

    {new_song}
    {ns}

Attributes may be added using [key/value
pairs](https://www.chordpro.org/chordpro/key_value_pairs/).

-   `toc=` *arg*\
    Add the song title to the table of contents if *arg*. This is the
    default case. Use `toc=no` to suppress this song from appearing the
    in the table of contents.
